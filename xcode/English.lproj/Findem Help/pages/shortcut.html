Shorcut Keys

This is a list of modifier keys for the program to make it even faster!

Lyrics Tab

  • ⌥ N -> Goto the next song
  • ⌥ Z -> Fetch lyrics
  • ⌥ S -> Search for lyrics on Google
  • ⌥ F -> Format the current lyrics
  • ⌥ A -> Apply the current lyrics
  • ⌥ - -> Remove select Album song from the panel
  • ⌥ H -> Show/Hide Album song panel

Song Queue

  • ⌥ G -> Get current selection from iTunes
  • ⌘ ⌥ - -> Remove a song from the Song Queue
